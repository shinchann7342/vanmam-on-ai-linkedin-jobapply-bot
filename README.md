## How to Run?

1. This app requires you to have Python 3 at least. Download the latest version of Python from [https://www.python.org](https://www.python.org).

2. You need to run `first_run.bat` and give it your credentials (don't worry, everything stays in your system; you can give your details).
   
   > **PS:** All your details will be saved in the `user-config.txt` file created from `first_run.bat` (**remember to delete it before sharing this app to someone if you're doing so locally**).

3. Once your details are captured in `first_run.bat`, this app should do its job (i.e., getting you a job ðŸ˜‚... aight, jokes apart). This app will open a Selenium window and execute the code which automatically applies for the LinkedIn jobs which has EasyApply enabled and no other jobs.

    > **PS:** I hope you have already applied for a few jobs on linkedin for which you might have filled in your details along with your updated resume, try to update these details on linkedin itself and check if you have your updated resume on LinkedIn before starting the app.

4. Don't worry, you don't have to run the `first_run.bat` every time. Instead, once your details are filled, you can run the `run.bat` file every time you want to run this application.

5. If the app fails or an error is encountered, check the following:
   
   * **5.1.** The job application limit is reached in LinkedIn (50 Applications per day). You can check it by logging in to your LinkedIn app and try to apply for a job. If an error throws, then try applying the next day (IK the market is bad, bro ðŸ¥² (bro is gender neutral ðŸ—¿) but you gotta wait 24 hours).
   
   * **5.2.** The app failed due to a network issue or something. Try restarting the app by executing the `run.bat` file. If it doesn't fix it, then pray ðŸ¥².

6. If you want the Telegram notifications for the jobs that got applied, then you have to fill in the chat ID and Bot token. ## Known Issues:
1. The app may sometimes get stuck on the application part, you can either restart the application manually or Use the Telegram Bot Functionality (ie. send "Restart" message from your Telegram Bot).

2. Same applies for ending the current execution (ie. send "End" message from your telegram bot to end the current execution)
